# Changelog — v0.7.0 → v0.7.2 (RAZUM fork)

Слияние upstream `fathah` v0.7.1 в форк РАЗУМ + гибрид-доработки.

## From upstream (v0.7.0 → v0.7.1)
- Новый хедер, центральная модалка slash-команд, model picker, анимация сайдбара
- Фиксы: удаление сессии (FK rollback), «Invalid time value»/1970-таймстампы, вложения
- SSH remote launcher hooks для профилей и расписаний
- Перестройка i18n, арабская локаль (+ RTL)

## RAZUM fork changes (v0.7.2)

### Fixed
- **Вход через Яндекс ID — в системном браузере.** OAuth открывается через
  `shell.openExternal`, а не во встроенном окне Electron: passport.yandex ломал
  вёрстку и не реагировал на клики в webview. Go-companion ловит redirect на
  loopback `127.0.0.1:18656` и завершает enroll. (`src/main/companion.ts`)

### Changed
- **Гибрид-гейтинг перенесён в новые модульные Settings-панели:** `ConnectionPane`
  скрывает тайлы Local/SSH в remote-режиме (сборка залочена на remote);
  `DataPane` гейтит баннер миграции OpenClaw за `connMode !== "remote"`.
- **Cron:** взята upstream-реализация `cronjobs.ts` (`parseCronListOutput`) +
  заново наложен наш гибридный dashboard-cron путь (`isRemoteOnlyMode` →
  `/api/cron/jobs` на `:9119` через companion-шим).
- RU остаётся локалью по умолчанию (`DEFAULT_ACTIVE_LOCALE=ru`); `ru` добавлен в
  `LANGUAGE_NATIVE_NAMES`.

### Notes
- 6 merge-конфликтов разрешены; CI зелёный (typecheck + 1553/1558 тестов; 2 падения
  — пред-существующий баг upstream `profiles.test`, не наш).
